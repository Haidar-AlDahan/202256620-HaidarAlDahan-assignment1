# SWE363 – Portfolio Assignment 1

This project is a simple, responsive personal portfolio website built for SWE363 (Web Engineering).

The goal of this assignment is to practice:
- HTML structure
- CSS layout and responsive design
- Basic JavaScript interactivity
- Responsible AI-assisted development

---

## Features

- Responsive layout (desktop, tablet, mobile)
- Dark / Light theme toggle (with localStorage persistence)
- Contact form with client-side validation (no backend)
- Clean folder structure and modular files
